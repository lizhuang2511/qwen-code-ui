我将通过以下步骤为程序界面添加图标：

1. **修改** **`main.py`** **文件**：

   * 新增 `get_icon_path` 函数，用于查找图标文件。它将优先查找 `frontend/dist/favicon.ico`，如果不存在则查找 `frontend/public/favicon.ico`。

   * 在 `webview.create_window` 调用中添加 `icon` 参数，传入找到的图标路径。

具体的代码变更如下：

在 `get_entry` 函数之前添加：

```python
def get_icon_path() -> str:
    paths = [
        os.path.join(BASE_DIR, "frontend", "dist", "favicon.ico"),
        os.path.join(BASE_DIR, "frontend", "public", "favicon.ico"),
    ]
    for p in paths:
        if os.path.exists(p):
            return p
    return ""
```

更新 `webview.create_window` 调用：

```python
    icon_path = get_icon_path()
    window = webview.create_window(
        "QWENCODE", 
        entry, 
        js_api=Api(), 
        text_select=True,
        width=1400,
        height=800,
        icon=icon_path
    )
```
