我将修改 `frontend/src/components/common/DirectoryPanel.tsx` 文件来实现右键新建文件的功能。

具体计划如下：

1.  **修改 `DirectoryPanel.tsx`**:
    *   **引入图标**: 从 `lucide-react` 引入 `FilePlus` 图标。
    *   **添加状态**: 添加 `newFileDialogOpen` (控制对话框显示), `newFileName` (新文件名), `newFileParent` (父目录路径) 等状态变量。
    *   **实现处理函数**: 添加 `handleCreateFile` 函数，该函数将调用 `api.write_file_content` 创建一个空文件。
    *   **添加右键菜单项**: 在 `ContextMenu` 中添加 "New File" (新建文件) 选项，点击后打开新建文件对话框。
    *   **添加对话框**: 在渲染部分添加一个 `Dialog` 组件，用于输入新文件的名称，类似于现有的 "New Folder" 对话框。

该方案遵循现有的代码风格，复用已有的 API (`api.write_file_content`)，并使用 React 状态管理 UI 交互。
